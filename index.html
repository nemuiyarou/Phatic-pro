LLM integrated web based student-teacher platform for language learning

A set of tools created to speed up learning, assist teachers, track progress, and provide feedback.

A Language Learning platform
Designed to augment teachers
Not replace them

Features

  • LLM (AI) integrated classrooms
  • Spoken level tracking
  • Fast and accurate material creation from web sources
  • Space-Repetition, context driven vocabulary study

Phatic is a fully fledged learning environment, for language teachers and students that integrates the latest LLMs to provide context, aid teacher feedback, and speed-up the learning process.
